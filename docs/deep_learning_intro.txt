딥러닝 개요

딥러닝은 인공신경망을 여러 층으로 쌓아 복잡한 패턴을 학습하는 머신러닝의 한 분야입니다. 뇌의 뉴런 구조에서 영감을 받았으며, 이미지 인식, 자연어 처리, 음성 인식 등 다양한 분야에서 뛰어난 성능을 보입니다.

퍼셉트론과 활성화 함수

단층 퍼셉트론은 선형 분류만 가능합니다. 다층 퍼셉트론(MLP)은 은닉층을 추가해 비선형 문제를 풀 수 있습니다. 활성화 함수는 비선형성을 도입합니다. ReLU(Rectified Linear Unit)는 음수를 0으로, 양수는 그대로 통과시켜 기울기 소실 문제를 완화합니다. Sigmoid는 출력을 0~1로 압축해 이진 분류에, Softmax는 다중 클래스 분류의 출력층에 사용합니다.

역전파(Backpropagation)

역전파는 출력 오차를 역방향으로 전파해 각 가중치의 기울기를 계산하는 알고리즘입니다. 연쇄 법칙(Chain Rule)을 이용해 각 레이어의 기울기를 계산합니다. 경사 하강법(Gradient Descent)으로 손실 함수를 최소화하는 방향으로 가중치를 업데이트합니다. Adam, SGD, RMSProp 등의 옵티마이저가 학습 속도와 안정성에 영향을 미칩니다.

손실 함수

회귀 문제에는 MSE(평균 제곱 오차), MAE(평균 절대 오차)를 사용합니다. 분류 문제에는 크로스 엔트로피 손실을 사용합니다. 이진 분류에는 Binary Cross Entropy, 다중 분류에는 Categorical Cross Entropy를 적용합니다.
